# Tee-Jay Multimedia - Finance Manager Pro

A comprehensive financial management system designed for multimedia service providers.

## Features

### 📝 Transaction Management
- Track income and expense transactions
- Categorize transactions by type (Video Production, Photography, Graphic Design, etc.)
- Add descriptions and personal notes to each transaction
- Real-time balance calculations

### 🪙 Petty Cash Tracking
- Manage small cash transactions separately
- Track cash additions and withdrawals
- Maintain detailed petty cash history

### 🧾 Invoice Generation
- Create professional invoices
- Track client information and service descriptions
- Preview invoices before sending
- Print and download capabilities

### 📊 Financial Analytics
- Monthly income vs expenses charts
- Category-based breakdowns for income and expenses
- Visual data representation
- Last 6 months of historical data

### 📋 Financial Reports
- Generate detailed monthly financial reports
- View income and expense breakdowns by category
- Calculate net profit/loss

### 💾 Data Management
- Export transactions to CSV format
- Local data persistence using browser localStorage
- Offline-first functionality with sync capability
- Online/offline status indicator

### 🌓 Theme Support
- Dark mode (default)
- Light mode toggle

## Getting Started

Simply open `index.html` in your web browser. All data is stored locally in your browser.

## Usage

1. **Add Transactions**: Go to the Transactions tab and fill in the form to add income or expense entries
2. **Manage Petty Cash**: Use the Petty Cash tab to track small cash transactions
3. **Create Invoices**: Go to Invoices tab to generate professional invoices for clients
4. **View Analytics**: Check the Analytics tab for visual representations of your financial data
5. **Generate Reports**: Use the Reports tab to create detailed financial summaries
